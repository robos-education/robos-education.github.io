### 코드 블록

인라인 코드 backtic(`` ` ``) 하나로 감싼다.
`print("hello")`

여러 줄 코드는 backtic 세 개로 감싼다. 언어 이름을 쓰면 색상이 표현된다.

```python
def hello():
    print("Hello Wodld~!")
```
